## Visual Design Notes

### Hero Section (Without Image):
- **Left Side**: Text content (title, description, badges, CTA button)
- **Right Side**: 
  - Option 1: Gradient background with pattern/texture
  - Option 2: Simple colored background with destination name
  - Option 3: Abstract geometric shapes/illustration
  - Option 4: Placeholder with "Image Coming Soon" text

### Color Scheme:
- Primary: Purple/Blue gradient (matching site theme)
- Background: White/Gray-50 sections
- Cards: White with subtle shadows
- Icons: Colored (blue, orange, green) matching section themes

### Typography:
- Heading: Large, bold (Poppins font)
- Body: Medium, readable (system font)
- Cards: Compact, scannable

### Spacing:
- Generous padding between sections
- Consistent card spacing
- Mobile-responsive grid layouts
